STEP 1: DATABASE SETUP (Estimated time: 10 minutes)
════════════════════════════════════════════════════
☐ Log in to Hostinger cPanel
☐ Go to "MySQL Databases" section
☐ Create new database:
   ☐ Name: enterprise_management
   ☐ Record the database name
☐ Create MySQL user:
   ☐ Username: ent_user (or your choice)
   ☐ Generate strong password (copy to secure location)
   ☐ Confirm password
☐ Add user to database with ALL PRIVILEGES
☐ In phpMyAdmin, select the database
☐ Click Import tab
☐ Upload database.sql file
☐ Click "Go" to execute
☐ Verify all 15+ tables were created successfully

═══════════════════════════════════════════════════

   ☐ $db_pass = "your_password" (your MySQL password)

☐ Save file

TROUBLESHOOTING QUICK REFERENCE
════════════════════════════════

ISSUE: 403 Forbidden
→ Check all files uploaded to public_html
→ Verify .htaccess file exists in both root and public/
→ Check file permissions (644 for files)

ISSUE: Database Connection Error
→ Verify MySQL credentials in config/database.php
→ Verify database exists in cPanel
→ Verify MySQL user has privileges
→ Test connection in phpMyAdmin

ISSUE: Login Fails
→ Verify admin user exists in database
→ Try default credentials: admin / admin123
→ Check user is_active = TRUE
→ Verify password hash in database

ISSUE: Blank Dashboard
→ Check browser console (F12) for errors
→ Verify API responses in Network tab
→ Check PHP error logs in cPanel
→ Verify database tables were imported

ISSUE: Module Pages Show "Coming Soon"
→ This is expected in this version
→ Dashboard and API are fully functional
→ Module UIs will be added in future updates
